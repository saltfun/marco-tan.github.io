SSH 快速登录

2020-02-03 1 min read # SSH

省略IP、用户名、端口

编辑~/.ssh/config文件

Host alias1
    HostName 6.6.6.6
    User root
    Port 22

Host alias2
    HostName 8.8.8.8
    User root
    Port 22

修改完成后使用ssh alias1，再输入密码即可登录

省略密码

将~/.ssh/id_rsa.pub配置到服务器的~/.ssh/authorized_keys文件中，即可直接通过ssh alias1登录

    • 省略密码
